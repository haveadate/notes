# linux 学习笔记

## 文件操作

### 创建文件

```shell
$ touch fileName
```

### 删除文件

```shell
$ rm fileName
```

### 查看文件

```shell
$ cat fileName
```

### 修改文件

```shell
$ vim fileName
```

## 目录操作

### 创建目录

```shell
$ mkdir dirName
```

### 删除空目录

```shell
# 只能删除空目录
$ rmdir dirName
# 删除目录
$ rm -rf dirName
```

### 陈列目录内容

```shell
ls
```
